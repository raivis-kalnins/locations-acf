# locations-acf
- Plugin Name: Locations ACF
-- Works for any ACF field, but designed for use with the Locations plugin.
-- Supports: Gutenberg blocks, Elementor, DIVI Builder, Bakery builder, and any other page builder that can render shortcodes.
-- Example Shortcodes: [wp_paginated_lp] => archive page, [wp_single_lp] => single location page, [loc_county] and [loc_city] for dynamic templates
